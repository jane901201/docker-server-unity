此專案的 .env 檔案要自己創建，格式如下

```
MYSQL_ROOT_USER=
MYSQL_ROOT_PASSWORD=
MYSQL_DATABASE=
MYSQL_HOST=
MYSQL_PORT=
```

遊戲本體的音檔(在Asset/Audio裡)由於檔案過大沒有上傳，需要的得自己去官方範例下載
